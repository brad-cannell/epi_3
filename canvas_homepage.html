Instructions for using the Course websites

Welcome to the Canvas website for PH2710 - Epidemiology 3. To complete this course, you will actually need to use a few additional websites as well. This may cause a little bit of confusion at first, but my hope is that it will allow you to access the best available tools for learning R programming and it will allow you to continue to access course resources long after the semester is completed.

Here is a list of the websites we will use in this course along with a brief overview of how we will use them:

  • Epi 3 Course Website: View the course outline, view individual learning modules, and access links to course resources

You should not have to do anything special to access the course website -- just click on the link above. I also recommend that you bookmark the course website in your browser of choice.

Using DataCamp

What is DataCamp?

DataCamp is an incredible resource for interactively learning to program in R, Python, and SQL. The website is broken into topic-specific courses that include videos, readings, and interactive coding sessions that provide you with immediate feedback about any errors in your code. Typically, you have to pay a monthly or yearly subscription fee to access DataCamp courses. However, DataCamp for the Classroom allows me to provide you with access to all of DataCamp's courses free of charge for 6 months. All you have to do is register!
